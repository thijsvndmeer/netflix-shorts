# Netflix Shorts

Netflix Shorts is a short-form "doomscroll" movie discovery application. Designed to feel like a modern vertical video feed, it allows users to discover movies through bite-sized preview clips, interact by liking/disliking/skipping, and jump straight into watching the full movie.

---
